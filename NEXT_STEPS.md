## Planned Features (v2.0+)

### Zone Customization
- Configurable zone ratios (e.g., 40/30/30 instead of equal thirds)
- Custom zone presets for different workflows
- Support for 2-zone and 4-zone layouts

### Keyboard Shortcuts
- Global hotkeys for snapping active window to zones
- Modifier key combinations (e.g., Ctrl+Option+1/2/3)
- Customizable key bindings in preferences

### Multi-Monitor Support
- Detect and manage multiple displays
- Per-monitor zone configurations
- Cross-monitor window movement

### Window Memory
- Remember window positions per app
- Restore layouts on app launch
- Save/load layout presets

### Visual Enhancements
- Zone preview overlay when dragging
- Animated snap transitions
- Dark/light mode adaptive UI

### Settings Panel
- Preferences window for all configuration
- Launch at login option
- Zone margin/gap settings
